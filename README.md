# esbuild-webpack-plugin

Use [esbuild](https://github.com/evanw/esbuild) as minifier for webpack.

## Install

```bash
$ yarn add esbuild-webpack-plugin
```

## Test

```bash
// Minify with terser
$ yarn build:example

// Minify with esbuild
$ yarn build:example:esbuild

// Do not minify
$ yarn build:example:nocompress
```
